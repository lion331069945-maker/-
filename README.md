# 永鼎股份 600105 实时量化看板

这是一个单页静态看板，当前版本 `v5`，可在永鼎股份 `600105.SH`、利通电子 `603629.SH`、华工科技 `000988.SZ` 之间切换监控。

## 功能

- 实时行情卡片：最新价、涨跌幅、成交量、日内区间
- 股票选择：支持永鼎股份、利通电子、华工科技
- 分时曲线：交易时段内按轮询数据逐步绘制
- 多因子量化买入信号：只提示信号，不自动交易
- 策略参数：均价突破、最小涨幅、放量倍数、RSI 上限、最大回撤、追高限制
- 行情源：优先新浪，失败后切换腾讯

## 策略框架

策略参考了 GitHub 开源量化社区常见的指标结构，例如 Qlib、vn.py、RQAlpha、Backtrader 等项目中常见的趋势、动量、成交量和风险控制思路。页面没有复制这些项目的代码，而是实现了适合单只 A 股盘中监控的轻量规则：

- 趋势确认：MA5/MA13、EMA8/EMA21、价格与短均线关系
- 动量确认：MACD 柱变化、RSI、相对开盘强度
- 成交量确认：最新成交增量相对近期均值的放量倍数
- 价格位置：动态均价偏离、布林带位置
- 风险过滤：涨幅、最大回撤、日内振幅、追高限制

## 部署

该项目是纯静态页面，适合直接部署到 GitHub Pages。

1. 推送到 GitHub 仓库。
2. 在仓库 `Settings -> Pages` 中选择 `Deploy from a branch`。
3. 分支选择 `main`，目录选择 `/root`。
