# 永鼎股份 600105 实时量化看板

这是一个单页静态看板，固定监控 A 股永鼎股份 `600105.SH`。

## 功能

- 实时行情卡片：最新价、涨跌幅、成交量、日内区间
- 分时曲线：交易时段内按轮询数据逐步绘制
- 量化买入信号：只提示信号，不自动交易
- 策略参数：突破动态均价、最小涨幅、放量倍数、追高限制
- 行情源：优先新浪，失败后切换腾讯

## 部署

该项目是纯静态页面，适合直接部署到 GitHub Pages。

1. 推送到 GitHub 仓库。
2. 在仓库 `Settings -> Pages` 中选择 `Deploy from a branch`。
3. 分支选择 `main`，目录选择 `/root`。

## 风险提示

本看板只作为行情观察和策略辅助工具，不构成投资建议，也不保证收益。
